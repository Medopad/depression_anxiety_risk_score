Installation
------------

To use Depression_Anxiety_Risk_Scores,

.. code-block:: console

   $ git clone ../depression_anxiety_risk_scores
   $ poetry install

-----

Depression_Anxiety_Risk_Scores's usage looks like:

.. code-block:: console
